# AI共創基盤 - ディレクトリ構成

## 全体構成

```
bookmark-rss-tyoukan/
├── services/              # バックエンドサービス
│   ├── auth/              # 認証基盤 (TypeScript/Hono)
│   ├── feed/              # フィード・記事・ブックマーク管理 (TypeScript/Hono)
│   ├── ai/                # AI記事選定・要約 (Python/FastAPI)
│   └── notification/      # 通知配信 (TypeScript/Hono)
├── apps/                  # クライアントアプリケーション
│   ├── web/               # WebUI (TanStack Start/React PWA)
│   ├── cli/               # CLI (Rust/clap)
│   └── extension/         # Chrome拡張機能 (WXT/TypeScript)
├── packages/              # 共有ライブラリ
│   ├── db/                # DBスキーマ・マイグレーション (Drizzle ORM)
│   └── mcp-server/        # MCPサーバー (Python/FastMCP)
├── infra/                 # IaC (AWS CDK)
├── openapi/               # OpenAPI仕様 (サービス間契約)
├── tests/                 # 結合テスト
├── docs/                  # 設計ドキュメント
├── scripts/               # ユーティリティスクリプト
├── Makefile               # 言語横断のタスクランナー
├── docker-compose.yml     # PostgreSQL 17 コンテナ
├── pnpm-workspace.yaml    # pnpmワークスペース定義
└── tsconfig.base.json     # TypeScript共通設定
```

## services/ — バックエンドサービス

### auth/ (TypeScript/Hono, Port 3000)

```
services/auth/
├── src/
│   ├── index.ts               # Honoアプリ + サーバー起動
│   ├── routes/
│   │   ├── me.ts              # GET /auth/me
│   │   ├── service-token.ts   # POST /auth/service-token
│   │   ├── jwks.ts            # GET /auth/.well-known/jwks.json
│   │   └── device.ts          # デバイスコードフロー (/auth/device/*)
│   ├── services/
│   │   └── service-account.ts # サービスアカウント管理・JWT署名
│   ├── middleware/
│   │   └── auth.ts            # 認証ミドルウェア
│   └── lib/
│       ├── db.ts              # DB接続
│       ├── logger.ts          # pino ロガー
│       ├── config.ts          # 環境変数設定
│       └── errors.ts          # カスタムエラー
├── __tests__/                 # ユニットテスト
├── package.json
└── tsconfig.json
```

### feed/ (TypeScript/Hono, Port 3001)

```
services/feed/
├── src/
│   ├── index.ts               # Honoアプリ + サーバー起動
│   ├── routes/
│   │   ├── feeds.ts           # フィードCRUD + RSS取得
│   │   ├── articles.ts        # 記事CRUD + 一括既読
│   │   ├── bookmarks.ts       # ブックマークCRUD + 全文検索
│   │   └── settings.ts        # ユーザ設定 + 通知対象一覧
│   ├── services/
│   │   ├── feed-service.ts    # フィード操作
│   │   ├── article-service.ts # 記事操作
│   │   ├── bookmark-service.ts # ブックマーク操作
│   │   └── rss-fetcher.ts     # RSS取得・パース
│   ├── middleware/
│   │   └── auth.ts            # JWT検証（ユーザー/サービス両対応）
│   └── lib/
│       ├── readability.ts     # HTML本文抽出
│       ├── opml-parser.ts     # OPMLインポート
│       ├── rss-discovery.ts   # フィードURL自動発見
│       ├── db.ts, logger.ts, config.ts, errors.ts
├── __tests__/
├── package.json
└── tsconfig.json
```
