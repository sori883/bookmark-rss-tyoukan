# AI共創基盤 - データモデル

## users

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| email | text | メールアドレス (UNIQUE) |
| name | text | 表示名 |
| email_verified | boolean | メール確認済み |
| image | text | プロフィール画像URL (nullable) |
| created_at | timestamp | 登録日時 |
| updated_at | timestamp | 更新日時 |

## sessions（Better Auth セッション管理）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| expires_at | timestamp | 有効期限 |
| token | text | セッショントークン (UNIQUE) |
| created_at | timestamp | 作成日時 |
| updated_at | timestamp | 更新日時 |
| ip_address | text | IPアドレス (nullable) |
| user_agent | text | ユーザーエージェント (nullable) |
| user_id | text | FK (users) |

## accounts（Better Auth OAuth アカウント管理）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| account_id | text | プロバイダ側のアカウントID |
| provider_id | text | プロバイダ識別子 (google等) |
| user_id | text | FK (users) |
| access_token | text | アクセストークン (nullable) |
| refresh_token | text | リフレッシュトークン (nullable) |
| id_token | text | IDトークン (nullable) |
| access_token_expires_at | timestamp | アクセストークン有効期限 (nullable) |
| refresh_token_expires_at | timestamp | リフレッシュトークン有効期限 (nullable) |
| scope | text | スコープ (nullable) |
| password | text | パスワードハッシュ (nullable) |
| created_at | timestamp | 作成日時 |
| updated_at | timestamp | 更新日時 |

## verifications（Better Auth 検証トークン管理）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| identifier | text | 検証対象の識別子 |
| value | text | 検証値 |
| expires_at | timestamp | 有効期限 |
| created_at | timestamp | 作成日時 (nullable) |
| updated_at | timestamp | 更新日時 (nullable) |

## jwks（Better Auth JWT プラグイン用鍵管理）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| public_key | text | 公開鍵 (JWK JSON) |
| private_key | text | 秘密鍵 (JWK JSON) |
| created_at | timestamp | 作成日時 |
| expires_at | timestamp | 有効期限 (nullable) |

## feeds

| カラム             | 型         | 説明         |
| --------------- | --------- | ---------- |
| id              | text      | PK         |
| user_id         | text      | FK (users) |
| url             | text      | フィードURL    |
| title           | text      | フィード名      |
| site_url        | text      | サイトURL     |
| last_fetched_at | timestamp | 最終取得日時     |
| created_at      | timestamp | 登録日時       |

## articles（RSS記事メタデータ）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| user_id | text | FK (users) |
| feed_id | text | FK (feeds) |
| url | text | 記事URL |
| title | text | タイトル |
| description | text | 記事概要（RSSのdescription） |
| is_read | boolean | 既読フラグ |
| published_at | timestamp | 公開日時 |
| created_at | timestamp | 保存日時 |
| updated_at | timestamp | 更新日時 |

## bookmarks（ブックマーク・本文あり）

| カラム              | 型         | 説明                                      |
| ---------------- | --------- | --------------------------------------- |
| id               | text      | PK                                      |
| user_id          | text      | FK (users)                              |
| article_id       | text      | FK (articles) nullable（RSS記事からのブックマーク時） |
| url              | text      | 記事URL (UNIQUE)                          |
| title            | text      | タイトル                                    |
| content_markdown | text      | 本文(Markdown)                            |
| search_vector    | tsvector  | 全文検索インデックス                              |
| created_at       | timestamp | 保存日時                                    |
| updated_at       | timestamp | 更新日時                                    |

## settings

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| user_id | text | FK (users) |
| webhook_url | text | Webhook URL |
| webhook_type | text | slack / discord |
| notification_hour | integer | 通知時間 (0-23, デフォルト 9) |
| created_at | timestamp | 作成日時 |
| updated_at | timestamp | 更新日時 |

## service_accounts（サービスアカウント）

authサービスが管理。サービス間認証用。各サービスが起動時に`client_id` + `client_secret`でサービスJWTを取得する。

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| service_name | text | サービス名（feed, ai, notification等） |
| client_id | text | クライアントID (UNIQUE) |
| client_secret_hash | text | ハッシュ化されたシークレット |
| created_at | timestamp | 登録日時 |

## notifications（通知履歴）

| カラム     | 型         | 説明                      |
| ------- | --------- | ----------------------- |
| id      | text      | PK                      |
| user_id | text      | FK (users)              |
| type    | text      | webhook                 |
| message | text      | 通知内容（要約+URL一覧を含むダイジェスト） |
| is_read | boolean   | 既読（WebUI通知履歴用）          |
| sent_at | timestamp | 送信日時                    |

## jobs（ジョブキュー）

| カラム | 型 | 説明 |
|---|---|---|
| id | text | PK |
| job_type | text | ジョブ種別（feed:fetch） |
| payload | jsonb | ジョブのパラメータ |
| status | text | pending / processing / completed / failed |
| created_at | timestamp | 登録日時 |
| started_at | timestamp | 処理開始日時 |
| completed_at | timestamp | 処理完了日時 |
| error | text | エラー内容（failedの場合） |

## インデックス

```sql
-- RSS記事フィルタ
CREATE INDEX idx_articles_feed ON articles(feed_id);
CREATE INDEX idx_articles_unread ON articles(is_read) WHERE is_read = false;
CREATE INDEX idx_articles_user ON articles(user_id);

-- ブックマーク全文検索
CREATE INDEX idx_bookmarks_search ON bookmarks USING GIN(search_vector);
CREATE INDEX idx_bookmarks_user ON bookmarks(user_id);

-- フィード
CREATE INDEX idx_feeds_user ON feeds(user_id);

-- ジョブキュー
CREATE INDEX idx_jobs_pending ON jobs(job_type, created_at) WHERE status = 'pending';
```
